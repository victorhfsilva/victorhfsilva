Hi! Welcome to my GitHub Repository. Here, you'll find a collection of projects, codes samples and experiments I performed along my journey.

## 🧑‍💼 About Me

I'm Victor Hugo Ferreira Silva, a Fullstack and Mobile software developer. My love for coding started during my exchange to Budapest where I studied Information Technology Engineering for one year. I'm graduated in Physics Engineering by Universidade Estadual do Mato Grosso do Sul. And I'm currently studying Eletronics and Telecommunication Engineering at Universidade Federal de Uberlândia. 

## 🔭 Current Work

I'm currently working as an Undergraduate Researcher on a mobile development project for agricultural applications. This project aims to develop an Android application to assist small-scale producers in autonomously classifying their batch of grains.

## 🌱 I’m Currently Learning...

- Mobile Development with Kotlin;
- Back End Development with Java and Kotlin;
- Front End Development with React.js;
- DevOps Practices;
- Among other subjects.
